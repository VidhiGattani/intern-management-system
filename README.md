# Intern–Project–Evaluation Management System

A full-stack web application built using:

- **ASP.NET Core Web API** (Backend)
- **Entity Framework + PostgreSQL** (Database)
- **React.js** (Frontend)

## Features
### **1. Intern Details**
- Search intern by ID
- View personal info, projects, and evaluations

### **2. Project Details**
- Search project by ID
- View project info and assigned intern + evaluations

### **3. Evaluation Filter**
- Filter evaluations by:
  - Score (min–max)
  - Evaluator (dropdown)
  - Date range
  - Project title keyword
- Shows matching interns + projects

## Tech Stack
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- React.js
- Axios
- CSS / Custom UI Components
